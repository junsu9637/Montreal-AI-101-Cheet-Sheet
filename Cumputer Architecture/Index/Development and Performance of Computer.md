## 컴퓨터의 세대별 발전

| 컴퓨터 세대 | 주요 소자 | 기억장치 | 특징 |
| :- | :- | :- | :- |
| 0세대 컴퓨터 | 기계적 릴레이 | 금속기어, 카드 | - |
| 1세대 컴퓨터 | 진공관 | 자기드럼 | 하드웨어 중점 개발, 큰부피와 높은 전력소모 |
| 2세대 컴퓨터 | 트랜지스터 | 자기코어 | 고급언어 등장 |
| 3세대 컴퓨터 | SSI, MSI | RAM, ROM | OS 개발, 다중 프로그래밍, 고급언어 활용 |
| 4세대 컴퓨터 | LSI, VLSI | SRAM, DRAM | 객체지향언어, 개인용 컴퓨터, 병렬 처리 기능 강화 |
| 5세대 컴퓨터 | VLSI,ULSI, GLSI, SoC | SDRAM, DRAM, flash | 비노이만 구조, 멀티코어, 유티쿼터스 |

### 1세대 컴퓨터 : 진공관 세대

```markdown
1. 높은 전력 소모로 인한 발열로 필수적인 냉각 장치로 인해 부피가 크다. 
2. 기억장치 : 자기드럼
3. 보조기억장치 : 종이 카드
4. 하드웨어 개발 중점에 기계어나 어셈블리어 사용
```

### 2세대 컴퓨터 : 트랜지스터 세대

```markdown
1. 전력문제 해결 및 신뢰도 확보
2. 기억장치 : 자기코어
3. 보조기억장치 : 자기드럼, 자기디스크
4. 고급언어 사용 및 시스템 소프트웨어 확립
```

### 3세대 컴퓨터 : 집적회로 세대

```markdown
1. 소형화 및 비용 절감
2. 기억장치 : 집적회로
3. 보조기억장치 : 자기디스크, 자기테이프
4. OS 개발 및 캐시 메모리 사용
```

### 4세대 컴퓨터 : 고밀도 집적회로 세대

```markdown
1. 소형화 및 비용 절감
2. 기억장치 : RAM
3. 보조기억장치 : 자기디스크, 광디스크
4. 객체지향언어 사용 및 네트워크 기술 발전
```
