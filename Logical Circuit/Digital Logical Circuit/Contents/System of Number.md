# Decimal Number

**10진수**는 기수가 10인 수로 0~9의 10개의 수로 표현한다.

바빌로니아인 60진법 -> 로마 5진법 -> 인도 10진법

# Binary Number

**2진수**는 기수가 2인 수로 0,1 두 개의 수로 표현한다.

펀치카드나 자기코어를 통해 0과 1을 표현했다.

# Octal Number and Hexadecimal Number

**8진수**는 기수가 8인 수로 0~7의 8개의 수로 표현한다.

**16진수**는 기수가 16인 수로 0~9의 10개의 수와 A~F 6개의 문자로 표현한다.
