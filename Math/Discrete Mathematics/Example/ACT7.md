### 파란 공 4개와 빨간 공 5개가 들어있는 항아리에서 임의로 한 개의 공을 꺼내는 경우 파란 공이 나올 확률은 얼마인가?
```markdown
확률은 계산하기 위해 나올 수 있는 모든 경우는 9개이다. 
그중에서 4개는 파란 공이기 때문에 파란 공이 나올 확률은 4/9이다.
```

### 두 개의 주사위를 던져서 나온 숫자의 합이 7이 될 확률은 얼마인가?
```markdown
한 개의 주사위를 던져 나올 수 있는 모든 경우는 6가지이다.
곱의 법칙에 의해 두 개의 주사위를 던져 나올 수 있는 모든 경우의 수는 6<sup>2</sup>이다.
그 중에서 합이 7이 되는 경우는 (1,6), (2,5), (3,4), (4,3), (5,2), (6,1)로 총 6가지이다.
따라서 두 개의 주사위를 던져서 나온 숫자의 합이 7이 될 확률은 6/36 = 1-6이다.
```

### 어떤 복권에서 사람들이 0~9 중에서 하나의 숫자를 선택하는 것을 4번 반복하여 차례대로 나열한 다음, 기계장치에 의해 임의로 선택된 4개의 숫자와 나온 순서까지 모두 일치하면 1등에 당첨되고, 같은 위치에 3개의 숫자가 일치하면 2등에 당첨된다. 1등과 2등에 당첨될 확률은 얼마인가?
```markdown
4개의 숫자가 나온 순서까지 일치하는 경우는 오직 한가지이다.
곱의 법칙에 의해 4개의 숫자를 선택하는 경우는 10<sup>4</sup>=10,000이다.
따라서 1등에 당첨될 확률은 1/10,000이다.

4개의 숫자 중에서 같은 위치에 3개의 숫자가 일치해야한다.
이것은 어떤 위치에 있는 1개의 숫자만 다르다는 것을 의미한다.
어떤 위치에 하나의 숫자만 다른 경우는 9이다.
숫자가 4개이기 때문에 합의 법칙에 의해 9+9+9+9=36이다.
곱의 법칙에 의해 4개의 숫자를 선택하는 경우는 10<sup>4</sup>=10,000이다.
따라서 2등에 당첨될 확률은 36/10,000이다.
```

### 1~n까지의 정수 중에서 6개를 골라 모두 맞추면 상금을 주는 복권이 있다. n이 40일 때 6개의 숫자를 모두 맟줄 확률은 얼마인가?
```markdown
당첨되는 경우의 수는 한 가지뿐이다.
숫자 40개 중에서 6개를 고르는 경우는 C(40,6)=40!/34!6!=3,838,380이다.
따라서 당첨될 확률은 1/3,838,380이다.
```

### 포커 게임에서 카드 5장을 받았을 때, 숫자가 같은 카드 4장이 들어올 확률은 얼마인가?
```markdown
카드 5장에서 숫자가 같은 카드 4장이 포함되는 경우는 곱의 법칙에 의해 다음을 곱한 것과 같다.
1. 13가지 숫자 중에서 하나를 고르는 경우 = C(13,1)
2. 고른 숫자에 4가지 무늬가 표시된 카드 4장을 고르는 경우 = C(4,4)
3. 나머지 한 개의 카드를 고르는 경우 = C(48,1)
카드 한 세트에서 5장을 고르는 경우는 C(52,5)이다.
따라서 조건에 만족하는 경우의 확률은 C(13,1)C(4,4)C(48,1)/C(52,5)이다.
13*1*48/2,598,960
```

### 포커 게임에서 5장의 카드 중 3장의 카드 숫자가 같고, 나머지 2장의 카드 숫자가 같은 경우를 풀하우스라 한다. 풀 하우스가 들어올 확률은 얼마인가?
```markdown
풀하우스는 곱의 법칙에 의해 다음을 곱한 것과 같다.
1. 13가지 숫자 중에서 2개의 숫자를 고르는 경우 = P(13,2)
2. 첫 번째 숫자가 적힌 4장의 카드 중에서 3장을 고르는 경우 = C(4,3)
3. 두 번째 숫자가 적힌 4장의 카드 중에서 2장을 고르는 경우 = C(4,2)
카드 한 세트에서 5장을 고르는 경우는 C(52,5)이다.
따라서 조건에 만족하는 경우의 확률은 P(13,2)C(4,3)C(4,2)/C(52,5)이다.
3,744/2,598,960
```

### 1~50까지 번호가 매겨진 50개의 공이 담겨 있는 바구니에서 5번 공을 꺼내서 나온 순서대로 나열한 번호가 11, 4, 17, 39, 23일 다음조건에 따라 구하면 얼마인가?

### 한 번 꺼낸 공을 바구니에 다시 집어넣지 않고, 다음 공을 꺼내는 경우(비복원 추출)
```markdown
곱의 법칙에 의해 5개의 공을 순서대로 꺼내는 경우는 50*49*48*47*46이다.
숫자가 순서대로 나오는 경우는 1가지이다.
따라서 1/50*49*48-47*46 = 1/254,251,200이다.
```


### 꺼낸 공을 바구니에 다시 집어넣고, 다음 공을 꺼내는 경우(복원 )
```markdown
곱의 법칙에 의해 5개의 공을 순서대로 꺼내는 경우는 50*50*50*50*50이다.
숫자가 순서대로 나오는 경우는 1가지이다.
따라서 1/50*50*50*50*50 = 1/312,500,000이다.
```


### 임의로 생성된 10개의 비트로 구성된 비트열에서 적어도 한 비트가 0일 확률은 얼마인가?
```markdown
10개의 비트 중에서 적어도 한 비트가 0인 사건을 E라고 하자.
E의 여사건은 모든 비트가 1인 사건이 된다.
표본공간 S는 길이가 10인 모든 비트열의 집합이다.
따라서 P(E) = 1-P(여집합) = 1-1/1024 = 1023/1024
```
